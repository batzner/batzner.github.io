Create a VAE¶

To keep this and future notebooks clean, we use a custom VAE implementation from src.vae. This takes 4 models - one for each of the mean / variance of the encoder / decoder. It simply wires them together with sampling layers and also manages the separate training of the decoder's variance estimate. In the beginning, all weights are trainable except those exclusive to the decoder's variance network.

In [3]:
from tensorflow.python.keras import Sequential, Model
from tensorflow.python.keras.layers import Dense, Input
from src.vae import VAE

original_dim = 2
latent_dim = 2

# Create the encoder models
enc_input = Input((original_dim,))
enc_shared = Sequential([
    Dense(10, activation='softplus'),
    Dense(10, activation='softplus')
])
enc_mean = Model(enc_input, Dense(latent_dim, activation='linear')(
    enc_shared(enc_input)))
enc_var = Model(enc_input, Dense(latent_dim, activation='softplus')(
    enc_shared(enc_input)))

# Create the decoder models
dec_input = Input((latent_dim,))
dec_shared = Sequential([
    Dense(10, activation='softplus'),
    Dense(10, activation='softplus')
])
dec_mean = Model(dec_input, Dense(original_dim, activation='linear')(
    dec_shared(dec_input)))
dec_var = Model(dec_input, Dense(original_dim, activation='softplus')(
    dec_shared(dec_input)))

vae = VAE(enc_mean, enc_var, dec_mean, dec_var, dec_stddev=0.1)

Train the VAE¶

Train the encoder and the decoder's mean network, but not the decoder's variance network. For now, we use a constant stddev of 0.1 for the decoder and train the variance estimation later.

Plot the reconstructions¶

Since the decoder's variance has not been trained yet, we only visualize the mean of each reconstruction's probability distribution.

In [6]:
def plot_reconstructions(use_encoder_mean, use_decoder_mean):
    # Sample from the encoder or just take the mean for better estimates
    if use_encoder_mean:
        encoded, encoded_mean, encoded_var = vae.encoder.predict(x_plot)
        decoded, decoded_mean, decoded_var = vae.decoder.predict(encoded_mean)
    else:
        decoded, decoded_mean, decoded_var = vae.model.predict(x_plot)
    
    reconstructions = decoded_mean if use_decoder_mean else decoded
    scatter_plot = go.Scatter(
        x = reconstructions[:, 0],
        y = reconstructions[:, 1],
        mode = 'markers',
        marker = {'color': y_plot, 'colorscale': colorscale}
    )
    data = [scatter_plot]
    iplot(go.Figure(data=data, layout=layout), config=config)
    
plot_reconstructions(use_encoder_mean=False, use_decoder_mean=True)

Visualize the latent space¶

We can now also plot a heatmap of the standard deviation $\log(\sum_{j=1}^D \sigma_j(z))$ for a grid of points $z$ in the latent space, as done in the paper. We will see that the variance estimates are poor.

In [9]:
# Display a 2D plot of the classes in the latent space
encoded, mean, var = vae.encoder.predict(x_plot)
scatter_plot = go.Scatter(
    x = mean[:, 0],
    y = mean[:, 1],
    mode = 'markers',
    marker = {'color': y_plot, 'colorscale': colorscale}
)

# Plot a heatmap showing the log of the sum of stddevs of the decoder
heatmap_z1 = np.linspace(-4, 4, 100)
heatmap_z2 = np.linspace(-4, 4, 100)
heatmap_grid_1, heatmap_grid_2 = np.meshgrid(heatmap_z1, heatmap_z2)
heatmap_grid_1 = heatmap_grid_1.reshape((-1, ))
heatmap_grid_2 = heatmap_grid_2.reshape((-1, ))
heatmap_feed = np.vstack([heatmap_grid_1, heatmap_grid_2]).T
_, _, heatmap_var = vae.decoder.predict(heatmap_feed)
heatmap_var = heatmap_var.reshape((100, 100, 2))

# Reduce to the log of the sum of stddevs
heatmap_values = np.log(np.sum(np.sqrt(heatmap_var), axis=-1))

heatmap = go.Heatmap(x=heatmap_z1, y=heatmap_z2, z=heatmap_values, 
                     colorscale=[[0.0, 'black'], [0.5, 'white'], 
                                 [1.0, '#71001f']])
data = [heatmap, scatter_plot]
iplot(go.Figure(data=data, layout=layout), config=config)

Plot the reconstructions¶

Finally, we can plot the reconstructions using only the means again. We see that during training the decoder's variance, the latent means and the corresponding reconstructions did not change in any way. This means, the weights were frozen and only the weights exclusive to the decoder's variance network were trained.

Conclusion¶

The VAE decoder's mean estimates on the moons dataset are good, but the variance estimates are poor. This will be addressed in the next notebook, 07-moons-rbf.ipynb.
