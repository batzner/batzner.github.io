Can Euklid tell a cat from a face?¶

We will load 2000 images of faces and compute the distribution of pairwise Euclidean distances among them. Then, we load 118 cat images, run each of them against every face image and compare the distribution of Euclidean distances with that of the faces themselves.

from io import BytesIO
from zipfile import ZipFile
from urllib.request import urlopen

import numpy as np
import matplotlib.pyplot as plt
from PIL import Image

faces_url = 'http://www.cfpw.io/cfp-dataset.zip'
image_path_template = 'cfp-dataset/Data/Images/%03d/frontal/%02d.jpg'
num_persons = 200
faces_per_person = 10

cats_url = 'https://github.com/maxogden/cats/archive/master.zip'
cat_image_path_template = 'cats-master/cat_photos/kublai%d.jpg'
num_cats = 118

In [3]:
faces = []
for i_person in range(1, num_persons + 1):
    for i_face in range(1, faces_per_person + 1):
        image_path = image_path_template % (i_person, i_face)
        with zipfile.open(image_path) as f:
            image = Image.open(f)
            image = image.resize((100, 100))
            faces.append(np.array(image))

faces = np.array(faces)
print('faces.shape: %s faces.dtype: %s' % (faces.shape, faces.dtype))

# Compute the pairwise distances
face_distances = [np.linalg.norm(faces[a] - faces[b])
                  for a in range(len(faces)) for b in range(a)]

plt.hist(face_distances, bins=200, density=True)
plt.xlim(0, 40000)

Conclusion¶

The distributions are only slightly different. The average cat image even seems to be closer to all face images than the average face image. Given the distribution of a single image at test time, it will therefore be hard to detect outliers / cats.
